EcoGlow Store

Proyecto final — Talento Tech 2.0

EcoGlow Store es un prototipo de tienda virtual enfocada en productos sostenibles, con una experiencia amigable, moderna y pensada para promover la compra responsable.

Objetivo del proyecto

Desarrollar la estructura visual (UI UX) de una tienda ecológica que cumpla con los requisitos del reto del módulo de Habilidades de Poder, integrando:

Storytelling del producto

Presentación clara y profesional

Uso correcto de herramientas digitales como Figma y GitHub

Características principales del prototipo

Pantalla inicial del e-commerce

Sección de productos

Identidad visual ecológica

Navegación simple y limpia

Tecnologías utilizadas

Figma (diseño UI/UX)

GitHub (repositorio del entregable)
